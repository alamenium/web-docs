position:
  • sticky -Sticks to the top of the nearest block-level ancestor

  • fixed -Removed from flow and positioned to initial containing block

  • relative -Stays in flow, positioned based on "top", "right" etc

  • absolute -Removed from flow, positioned relative to nearest POSITIONED ancestor, based on "top", "right" etc

  • static -Positioned based on normal flow.. "top", "right" values are ignored

Float:
  • left -object will float to the left

  • right -object will float to the right

  • inline-start -object float to the start of the text (left with ltr & and right with rtl text)

  • inline-end -object float to the end of the text (right with ltr text & left with rtl text)

Float removes the element from the website flow, and the width must be specified or else it will assume the full width of containing element
