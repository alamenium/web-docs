justify-content

x-axis parent
  • space-between: distributes items evenly, first & last items are flush with the wall
  • space-around: Distribute items evenly Items have a half-size space on either end
  • space-evenly: Distribute items evenly Items have a equal-size space on either end
  • flex-start: packs flex items from the start
  • flex-end: packs flex items from the end
  • left/right/center: packs flex items left/right/centert

align-items

y-axis parent
  • stretch: flex items with no height constraints are stretched to fill y-axis
  • center: Packs items to the center (vertically)
  • flex-start: Packs flex items from the start
  • flex-ennd: Packs flex items from the end
  • baseline: All flex items are aligned such that their flex container baselines align
align-self

y-axis child
  • Same values as align-items, applies to specific child

  • auto, applies the same value of align-items of the parent!

align-content

y-axis parent
  • space-between: distributes rows evenly, first & last items are flush with the wall
  • space-around: Distribute rows evenly Items have a half-size space on either end
  • space-evenly: Distribute rows evenly Items have a equal-size space on either end
  • flex-start: packs flex rows from the start
  • flex-end: packs flex rows from the end
  • center: packs flex rows to the center
  • baseline/first baseline/last baseline: packs flex rows according to their baseline
