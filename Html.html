{/Html}

HTML (HyperText Markup Language) is used to give content to a web page and instructs web browsers on how to structure that content

Main tags Text tags List tags Media tags Table tags Attributes
Main Html Tag
Tag Description
!DOCTYPE html Tells the browser the type of document!
!-- and closing -- Content inside of it will be commented out!
html Defines the document as html
head contains general info about the html page
title The title displayed in the browser's tab
link uses- href: to specify URL, rel: to specify relationship, type: to specify content type
body represents the content of the web page
nav encapsulates the pages main navigation links
main encapsulates the pages main section
div a container that divides the html document into sections
a used to create hyperlinks in html document
section defines elements such as chapters, headings or any other area with same theme
aside additional info that enhances another element but isn't needed
footer has contacts, copywrite info and sources
Text tags
h1> - <h6 Provides 6 different levels of heading elements
p Paragraph element which contains and displays a block of text
br will breate a line break (has NO closing tag)
em emphasizes text (most web browsers italicize it)
strong browsers will render it in bold by default
span Creates inline text, used to add styling to specific text within the same line
List element tags
ol creates an ordered list (numbered)
ul creates an unordered list (no numbers)
li creates a list element (put inside ol / ul)
Media tags
video Example: <video src="text.mp4" controls> Video not supported </video>

the controls atrribute tells the browser to add controls

the text between the brackets is displayed if the video couldn't load!
img Example: <img src="image.jpg" </img>

The img attribute is an empty attibute, meaning it doesnt need a closing tag!
audio Example:
<audio src="sfx.mp3" controls loop>
<a href = "fallback.mp3"> Download Audio </a>
</audio>

The anchor inside the audio tag is a fallback link in case the audioPlayer couldnt load
embed Can be used to implement any type of media, and its a self closing tag!
Example:
<embed src="media.gif"/>
figure Encapsulates media such as images, videos, audio files
figurecaption used to describe the media inside the figure element
Table Tags
table creates the main table element which will encapsulate the cells
thead defines the headings of table columns encapsulated in table rows
tbody Contains all the table data other than the heading and the footer
tfoot used to summarize table content at the end of the table
tr used to add rows to the table, is nested inside <thead>, <tbody> or <tfoot>
th nested inside the <tr> element and used to create a cell that will be the head of a group of cells (can be a title for rows or columns)
td can be nested inside the table row element to add a cell of data
Main Attributes
class used to group alot of elements together to apply styling
id used to give styling to a specific element
alt Provides alternative text for <img> in case the browser fails to render the image.
target Used in anchor element <a> to specify where the browser should open the link
Example:
target ="_blank" will open the link in a new tab!
rowspan used to specify how many rows (height) a single cell should take in a table
colspan used to specify how many columns (width) a single cell should take in a table
