{/The Box-Model}

Propety Possible values Demo
margin:
  • 30px -Sets margin from all sides
  • 20px 10px -Sets margin for y and x axis
  • 30px 20px 10px 15px -Sets margin for top, right, bottom, left
  • auto -Centers the object in the middle
  • 30px auto -gives 30px y axis margin, centers on x-axis
Margins collapse along the y axis, not x axis
margin:auto demo
overflow:
  • visible -content is rendered outside padding box
  • hidden -content outside padding box is hidden, has no scroll support except if done programmatically
  • clip -similar to hidden, but has no support for programmatic scrolling too
  • scroll -if content is bigger than box, a scroll bar appears
  • auto -looks like "visible" if it fits, or "scroll" otherwise
You can also set overflow for each axis separetely , for example:

overflow: hidden visible;
