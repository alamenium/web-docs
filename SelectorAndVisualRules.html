Selectors Typography

{/Selectors}

Item Description Demo
Type selector used to match elements of all given types 
    p{
          color:red;
    }
class selector Class selectors are denoted with a period, they are re-usable and may be applied to many elements 
    .column{
          color:red;
    }
ID Selector ID selectors should be unique and can be applied to only one element 
    #iphone{
          color:yellow;
    }
CSS Selector groups Makes multiple css selectors match the same css rule! 
    h1, h2{
          color:blue;
    }
Chaining selectors CSS rules can be set so that rule-sets apply to only such criteria 
h1.yellow{
    font-size: 2em;
    color: yellow;
}
//now only h1 elements with class "yellow" will be affected
descendant selector Used to match elements that are children of other elements! 
article p{
    color: yellow;
}
//The p children of article get affected!
selector specifity When there are multiple conflicting property values that point to the same element, the one with highest specifity gets applied 
    h1{
          color: yellow;
    }

    h1#header{
          color: red;
    }
//The h1 with id #header will be red, not yellow
!Important rule The !important rule is used on a declaration to override any other declaration 
    h1#header{
          color: red;
    }
    h1{
          color: yellow !important;
    }
//Even tho (h1) is less specific, the !important will override all the other color declarations and the text will be yellow!
